# CasinoRadar - Melhores Cassinos Online no Brasil

Site de reviews e rankings de cassinos online com foco em segurança, bônus, Pix e jogo responsável.

## Características

- 200+ cassinos analisados
- Filtros inteligentes por categoria, bônus e depósito
- Busca em tempo real
- Design responsivo e moderno
- Carregamento rápido (HTML/CSS/JS puro)
- SEO otimizado

## Estrutura

- `docs/index.html` - Site principal (GitHub Pages)
- `package.json` - Metadados do projeto
- `.gitignore` - Arquivos ignorados

## Como executar localmente

```bash
python3 -m http.server 8000 --directory docs
```

Acesse: http://localhost:8000

## Deploy

O site é automaticamente publicado no GitHub Pages quando há push para a branch `main`.
