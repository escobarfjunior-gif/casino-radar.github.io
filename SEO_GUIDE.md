## 📈 Métricas para Monitorar

### Google Analytics 4
```
Métricas Principais:
- Sessões por fonte (Orgânico, Direto, Social)
- Taxa de rejeição (Bounce Rate)
- Tempo médio na página
- Conversões (Cliques em "Ir para Cassino")
- Páginas mais visitadas
```

### Google Search Console
```
Monitorar:
- Impressões (quantas vezes aparece nos resultados)
- Cliques (quantas vezes clicam no seu link)
- CTR (Click-Through Rate)
- Posição média (qual posição aparece)
- Erros de rastreamento
```

### Microsoft Clarity (Gratuito)
```
Heatmaps e Gravações:
- Onde os usuários clicam mais
- Onde eles desistem
- Padrões de navegação
- Problemas de UX
```
